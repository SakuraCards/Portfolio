Para construir este jogo em Python, utilizei a biblioteca Pygame para criar a interface gráfica e gerenciar os eventos de entrada do usuário. A biblioteca OS foi usada para lidar com as diferentes plataformas de sistema operacional, garantindo que o jogo funcione em diferentes ambientes.
Além disso, utilizei a biblioteca Random para gerar obstáculos aleatórios em diferentes alturas, tornando o jogo mais desafiador e imprevisível. O jogo também possui uma função de pontuação que é atualizada a cada vez que o jogador desvia com sucesso de um obstáculo.
